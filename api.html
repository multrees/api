                    get /payments

                    Retrieve all payments.

                    • Response

                    HTTP status code 200

                    Body

                    Media type: application/json

                    Type: array of Payment

                    Items: Payment

                    • customerId: required (string)
                    • accountName: required (string)
                    • notice: (date-only)

                      If this is an advanced notice then this is the settlement date. Otherwise it is the date of the payment.

                    • repeat: (string)

                      If this is a regular payment then describe the repeat.

                    Examples:

                    {
  "customerId": "TODO",
  "accountName": "TODO",
  "repeat": "first day of month",
  "notice": "2017-02-28"
}
